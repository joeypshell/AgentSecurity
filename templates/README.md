# Operational Templates

Last reviewed: 2026-07-08

These templates convert the guidance in `docs/08-enterprise-copilot-mcp-action-plan.md`, `docs/09-enterprise-control-matrix.md`, `docs/10-mcp-server-admission-standard.md`, and `docs/11-red-team-validation-plan.md` into repeatable intake and review artifacts.

| Template | Use |
| --- | --- |
| `copilot-agent-pilot-intake.md` | Request a controlled pilot for VS Code agent mode, Visual Studio agent mode, cloud/coding agent, or MCP. |
| `mcp-server-admission.md` | Review and approve an MCP server before it is added to a registry or allowlist. |
| `agent-security-exception.md` | Time-box an exception for write-capable tools, local MCP, preview features, or sensitive repositories. |
| `red-team-test-case.md` | Record a validation scenario and evidence for agent/MCP controls. |

Prefer copying these into your ticketing/GRC system rather than treating repository Markdown as the system of record.
